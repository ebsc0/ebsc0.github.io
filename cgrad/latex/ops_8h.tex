Multiply two matrices. 

\doxylink{struct_matrix}{Matrix} multiplication algorithm naively implemented using forloops. An improvement would be using Winograd algorithm or Strassen\textquotesingle{}s algorithm. col size of a must equal row size of b. The resulting matrix will have the same row size as a and the same col size as b.

Calculate the cross correlation matrix. 

Cross-\/correlation of a 2D matrix with a 2D kernel is similar to the convolution of a 2D matrix with a 2D kernel. The difference is, convolution rotates the kernel matrix by 180deg before performing the operation. Py\+Torch uses cross-\/correlation for their Conv2D as it is simpler and has no effect on performance.
