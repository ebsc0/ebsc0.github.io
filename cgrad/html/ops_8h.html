Calculate the cross correlation matrix.

Cross-correlation of a 2D matrix with a 2D kernel is similar to the convolution of a 2D matrix with a 2D kernel. The difference is, convolution rotates the kernel matrix by 180deg before performing the operation. PyTorch uses cross-correlation for their Conv2D as it is simpler and has no effect on performance.
